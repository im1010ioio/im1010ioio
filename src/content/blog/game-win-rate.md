這是一個簡單的遊戲勝率計算器，輸入您想要的勝率，就會告訴您需要再連續贏幾場。

這個小工具是很久以前我在玩 Garena 傳說對決時練習寫程式而做的。

🚀 沒想到，現在已經在 [CodePen 上累計獲得超過 26 萬次瀏覽](https://codepen.io/im1010ioio/pen/BwmYrx)，而且搜尋「勝率計算」的話，在台灣 Google 搜尋結果中排名第一或第二。

最近，我將它升級到 Vue 3，搬家到 GitHub Pages 上，還加了多語系。

雖然我現在已經不常玩塔防遊戲了，但希望這個工具能幫助仍在遊戲世界中奮鬥的玩家們，不論是英雄聯盟 、傳說對決或是其他類似的遊戲。
